# pokedex

>  Vue.js つかってポケモン図鑑を作ろうっていうあれこれ

## Build Setup
git clone https://github.com/s17001/vuePokeDex.git <br>
cd vuePokeDex <br>
npm install <br>
npm run dev <br>

### 主に使ってるやつ
pokeapi <br>
vue-chartjs <br>
VueSingleSelect  <br>
